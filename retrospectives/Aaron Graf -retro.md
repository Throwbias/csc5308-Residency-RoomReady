# Aaron Graf – Individual Retrospective

## 1. What was the hardest decision your team made today, and how did you resolve it?

Most of the decisions we made today were not too difficult, thankfully. The hardest decision was probably choosing what we wanted to create for the sprint project. It was early on a Saturday morning after we had all worked late the night before, so we were tired and realistic about our limits. That actually helped put us in the same mindset. We knew we did not want to choose an overly complicated project that would be difficult to finish in the short time available. We resolved this by choosing a product idea similar in structure to the group project we had already been working on during the semester. That gave us a familiar starting point while still allowing us to create a new project with a different purpose and audience.

## 2. How did using AI tools change your workflow? What did you have to be careful about?

AI tools were incredibly helpful because they allowed us to move faster with brainstorming, documentation structure, and technical troubleshooting. They helped reduce some of the time pressure by giving us a starting point for project artifacts and implementation ideas. However, I had to be careful when writing prompts to make sure I included enough context. If I left out an important detail, the AI could return information that did not match our actual project. I also had to be careful to use AI as a learning aid rather than relying on it so heavily that I would not understand the final work. Since we may need to explain our code and decisions during the presentation, understanding what we committed was important.

## 3. What would you build next if you had two more hours on the POC?

If we had two more hours, I would want to add features beyond basic room reservation, such as checking out computer equipment, HDMI cables, or other helpful library tools. I would also like to add a clearer staff-facing view so library staff could quickly see which rooms are reserved and which support items have been requested.

## 4. How does today connect to what you will be doing in Weeks 1–4 of the semester?

Today directly connects to what we have been doing in the first weeks of the semester. This sprint felt like a significantly accelerated version of the larger group project we have been working on together. We had to define a problem, narrow the scope, create documentation, make technical decisions, build a proof of concept, and prepare to explain our work as a team.
