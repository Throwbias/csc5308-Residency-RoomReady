## Elevator Pitch
Problem Statement

Library patrons often need quiet study spaces, but it can be difficult to know which rooms are currently available without walking through the library, asking staff, or relying on outdated signage. This creates wasted time for patrons and unnecessary interruptions for library staff.

Solution

RoomReady addresses this problem by giving patrons a simple kiosk-style webpage where they can view available library study rooms and reserve an open room immediately. For the proof of concept, the system focuses on current room availability only. It does not include future scheduling, check-in, recurring reservations, or complex calendar management.
